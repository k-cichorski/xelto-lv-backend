# xelto-lv-backend
Backend for Xelto Log Viewer
## Installation
```bash
git clone https://github.com/k-cichorski/xelto-lv-backend
```
Go to project directory and run
```bash
npm install
```
Open .env.example file, set the variables and save the file as .env
## Usage
Go to project directory and run
```bash
npm start
```
